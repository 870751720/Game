# Game

基于 Phaser 3 + TypeScript + Vite 的 2D 网页游戏项目。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build
```

## 目录结构

```
├── index.html
├── vite.config.ts
├── src/
│   ├── main.ts
│   ├── gameConfig.ts
│   └── scenes/
│       └── BootScene.ts
└── public/assets/     # 游戏资源
```

## 技术栈

- [Phaser 3](https://phaser.io/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
