# 战斗系统框架

> 本文档定义战斗类系统的装备规则与插槽机制。

---

## 核心规则

1. **默认只能装备一个战斗系统**。所有持有者初始只有一个战斗系统插槽。
2. **每个战斗系统可独立升级**。通过战斗使用获得经验，提升系统等级，解锁新技能或强化现有技能。
3. **额外插槽极其稀有**。插槽不会随普通成长自动增加，而是作为特殊奖励投放（如击败特定Boss、完成传说任务、探索隐秘遗迹）。
4. **插槽数量 = 同时装备的战斗系统数量上限**。获得一个额外插槽，意味着可以同时激活两个战斗系统的能力。

---

## 设计意图

- 让"同时持有多个战斗系统"成为稀缺且令人兴奋的时刻。
- 单个战斗系统本身足够完整，不需要拼凑多个才能玩。
- 红狼之所以是传奇，可能因为他拥有远超常人的插槽数量。
