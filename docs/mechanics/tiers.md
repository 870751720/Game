# 品级体系总论

> 本文档定义游戏内所有可分级事物的统一品级标准。

---

## 品级梯度

| 颜色 | 名称 | 定位 |
|:----:|------|------|
| 白 | **普通** | 随处可见，基础品质 |
| 蓝 | **精良** | 稍有门槛，普通冒险者主力 |
| 紫 | **卓越** | 出众，精英阶层持有 |
| 金 | **稀世** | 世间少有，强者才有资格拥有 |
| 红 | **无双** | 绝无仅有，可能改变势力格局 |

---

## 设计原则

1. **统一标准**：本套品级适用于装备、材料、技能、怪物、系统等所有可分级事物。
2. **颜色为底层标识**：UI、掉落光效、提示文本等均以颜色为第一辨识度。
3. **名称是世界观层级的称呼**：游戏内NPC、文献、黑市交易等场景使用上述名称。
4. **品级决定上限，不决定内容**：同品级事物可能天差地别，品级只标识其规则容器的质量或稀有程度。
