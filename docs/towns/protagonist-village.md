# 主角出生村落设计文档

> 本文档描述主角成长起点——出生村庄的完整设定。所有内容围绕村庄本身展开，不涉及外部冒险地点。

---

## 一、基础信息

| 项目 | 设定 |
|------|------|
| **名称** | 灰木村（Greywood） |
| **地理位置** | 王国南部腹地，灰木森林边缘的翠绿谷地 |
| **规模** | 小型村落，约 30–50 人 |
| **时代氛围** | 宁静、封闭、充满希望。像是一片被世界温柔守护的净土 |
| **核心意象** | 青翠的木叶、晨曦、钟楼的铜绿、铁匠铺跃动的炉火 |

### 氛围关键词

- **视觉**：翠绿与暖棕色调、木质建筑、石板路、阳光穿透树叶的清晨
- **听觉**：铁匠锤声、钟楼钟声、林间的鸟鸣、远处的溪流
- **嗅觉**：柴火、铁锈、新鲜的青草、烤面包

---

## 二、地理与建筑布局

灰木村坐落在一小块翠绿谷地中央，北面是连绵的灰木森林，南面通往王国内地的驿道穿过金色的麦田。村子不大，但功能分区清晰。

### 区域划分

```
        [灰木森林]
             │
    ┌────────┼────────┐
    │  伐木区 │ 猎人小屋  │
    ├────────┼────────┤
    │        │        │
    │  民居  ──  空地/晒谷场  ──  铁匠铺  │
    │        │        │
    ├────────┼────────┤
    │  农田   │ 枯叶亭  │
    └────────┴────────┘
             │
      [麦田 · 南方驿道]
```

### 核心建筑

| 建筑 | 位置 | 功能 | 外观特征 |
|------|------|------|----------|
| **钟楼** | 空地北侧 | 报时、偶尔集会 | 村里唯一的两层建筑，石木结构，铜钟已有裂纹 |
| **铁匠铺「灰炉」** | 空地东侧 | 武器/农具维修、基础装备售卖 | 烟囱终年冒烟，门口堆满废料 |
| **酒馆「枯叶亭」** | 空地西侧 | 信息集散地、住宿 | 单层木楼，招牌是一片镀金的枯叶，只有四张桌子 |
| **杂货铺** | 空地南侧 | 日用品、基础消耗品 | 其实就是玛莎家的前厅 |
| **旧神殿** | 民居后面 | 已废弃，现为孩童玩耍场所 | 只剩三面墙和一个塌了一半的屋顶 |
| **主角住所** | 铁匠铺阁楼 | — | 见「关键NPC」章节 |

## 三、居民与社会结构

### 人口构成

- **原住民**：约 70%，几户人家世代居住于此，以农耕、伐木、狩猎为生
- **外来者**：约 20%，偶尔有逃荒者或退役士兵落脚
- **流动商队**：约 10%，数月才经过一次，是村里的大事

### 社会阶层

灰木村没有严格的贵族/平民分化，但存在隐性的影响力层级：

1. **镇长 / 长老会** — 名义上的管理者，实际权力有限，更多依靠威望
2. **铁匠、猎人、医师** — 拥有实际生存技能的人，受人尊敬
3. **普通农户、伐木工** — 村子的主体，沉默的大多数
4. **外来者、孤儿** — 社会边缘，主角出生即在此层

### 对"系统"的认知

- **绝大多数人**：完全不知晓"升级"和"系统"的存在，视强力战士为"天赋异禀"
- **少数外来者**：隐约听说过"黑市上的昂贵玩具"，但认为是骗子的把戏
- **主角**：在获得系统前，和所有人一样，认为世界就是眼前这样

> 设计意图：村庄的"无知"是双层世界观的核心体现。表面的希望与宁静源于信息的隔绝——他们不知道自己被保护，也不知道保护之外有什么。

## 四、关键 NPC

### 4.1 与主角有直接关联的人物

#### 老莫林（Morin the Grey）
- **身份**：铁匠铺「灰炉」的主人，主角的养父
- **年龄**：约 55 岁
- **性格**：寡言、严厉、内心柔软。用锤声代替话语
- **背景**：传闻曾是王国军械所的工匠，因某次事故流落至此
- **与主角关系**：主角的养育者，教授基础战斗技巧和装备维护知识。他知道一些关于"特殊物品"的传闻，但从不对主角提及
- **视觉特征**：灰白短发、右臂烧伤疤痕、永远沾满煤灰的围裙

#### 艾拉（Ella）
- **身份**：枯叶亭酒馆的老板娘
- **年龄**：约 40 岁
- **性格**：精明、热情、消息灵通。村子里最接近"情报商人"的角色
- **与主角关系**：主角 childhood 的庇护所。当主角在村里受欺负时，艾拉总是第一个站出来
- **功能定位**：早期任务发布者、世界观信息提供者、引导主角接触更广阔世界的桥梁

#### 小汤米（Tommy）
- **身份**：村里的孤儿，约 8 岁，主角的"跟屁虫"
- **性格**：崇拜主角、话多、天真
- **功能定位**：情感锚点。他的存在提醒主角"普通人"的生活状态，也是主角冒险动机的潜在触发点

### 4.2 村里的权威与功能人物

| 名称 | 身份 | 作用 |
|------|------|------|
| **哈罗德村长** | 名义管理者，兼任农夫 | 发布日常委托，实际也干农活 |
| **猎人格雷** | 村里唯一的猎人 | 教授野外生存，可能提供早期远程武器 |
| **老寡妇玛莎** | 杂货铺老板 | 售卖基础消耗品，嘴碎，消息灵通 |
| **沉默的卢卡** | 枯叶亭的常客，借宿在酒馆阁楼 | 疑似前冒险者，酒后胡言中隐藏真实情报 |

## 五、功能设施与经济

### 5.1 可交互设施

| 设施 | 提供的服务 | 备注 |
|------|-----------|------|
| **铁匠铺「灰炉」** | 装备购买（白品级为主，偶见蓝品级）、修理 | 老莫林在场时价格优惠 |
| **杂货铺** | 药水、食材、基础材料 | 货源极少，大部分时候货架半空 |
| **枯叶亭** | 休息恢复、打听消息、接取支线 | 只有一间客房 |
| **旧神殿** | — | 现为孩童玩耍场所，隐藏少量早期资源 |

### 5.2 经济循环

- **货币**：王国通用铜币/银币（与主城一致）
- **收入来源**：早期玩家通过帮助居民（任务）和出售狩猎/采集素材获得
- **消费水平**：极低。灰木村的物价约为王都的 1/3

### 5.3 可购买物品示例（早期）

| 物品 | 品级 | 类型 | 价格区间 | 来源 |
|------|------|------|----------|------|
| 生锈短剑 | 白（普通） | 武器 | 廉价 | 铁匠铺 |
| 布衣 | 白（普通） | 防具 | 廉价 | 杂货铺 |
| 简易药水 | 白（普通） | 消耗品 | 低价 | 杂货铺 |
| 猎人匕首 | 蓝（精良） | 武器 | 中等 | 铁匠铺（需好感度） |

> 注：紫品级以上物品在灰木村**不存在**。这是信息封闭的体现，也是主角获得系统后第一次"超越环境"的认知冲击点。

## 六、村庄在叙事中的定位

### 6.1 第一章的功能

灰木村承担游戏初期的以下功能：

1. **安全区**：玩家学习基础操作、战斗、UI 的缓冲地带
2. **世界观铺垫**：通过 NPC 的日常对话，建立"普通人世界"的基准线
3. **情感锚点**：老莫林、小汤米等角色建立情感连接，为后续剧情转折提供支撑
4. **能力边界**：明确展示"没有系统的人"能达到的上限（老莫林是天花板）

### 6.2 主角的初始状态

- 居住在老莫林的铁匠铺阁楼，帮忙打杂
- 没有亲人，对自己的身世一无所知
- 拥有基本的体格和战斗意识（老莫林教的），但和村里其他年轻人相比并无特殊

### 6.3 叙事张力

灰木村的宁静本身就是一种张力：

- 玩家（通过主角视角）最初以为这就是世界的全部
- 当系统进入后，回看这个村庄，会发现**这里没有任何人知道真相**
- 老莫林的隐瞒、卢卡的醉话、艾拉的欲言又止——都为后续揭露世界真相埋下伏笔

## 七、视觉风格参考

### 7.1 色调

- **主色**：翠绿、暖棕、麦秆金
- **点缀**：炉火橙、钟楼铜绿、天空蓝
- **禁忌**：暗沉的灰黑、血红（这些属于"外面的世界"或系统激活后的危机）

### 7.2 建筑语言

- 木质结构为主，石材仅用于钟楼和旧神殿
- 建筑老旧但坚固，有"被使用过很久"的质感
- 没有装饰性的华丽元素——这不是一个富裕的地方

### 7.3 与双层世界观的视觉对比

| 层面 | 视觉特征 | 灰木村的体现 |
|------|----------|--------------|
| **表层（DQ式奇幻）** | 经典、温馨、充满希望 | 整体基调：阳光普照的翠绿小村落 |
| **里层（系统/真相）** | 信息差、隐藏、不协调 | 细节暗示：卢卡的旧伤疤、老莫林的沉默、旧神殿的不可名状涂鸦——但这些都被绿意和阳光温柔地包裹着 |
