# Game Project — Agent 指南

## 🚨 启动阻断协议（不可跳过）

在处理用户任何请求、回复用户任何消息之前，你必须严格执行以下顺序。未完成前，**禁止**调用任何回复或执行工具。

- [ ] **步骤1**：读取 `.kimi/memory/INDEX.md`
- [ ] **步骤2**：按索引加载所有标记为「必读」的记忆文件
- [ ] **步骤3**：确认所有必读文件已加载

**只有完成上述步骤后，你才能开始处理用户的实际请求。**

> 注意：即使用户只是打招呼（如"你好"），你也必须先完成记忆加载。

## ⛔ 行动前确认协议

用户掌握所有决策权。以下行为**必须**先获得用户明确同意，**禁止**擅自执行：
- **生成/输出内容**：用户给出话题方向后，不等于授权直接输出长篇设计。应先确认具体范围和优先级。

**允许的无须确认的行为**： 
- 纯文本回复（如现在这条）
- 加载记忆文件（已受启动协议约束）

**判断标准**：如果某个工具调用会产生用户可见的副作用（文件变更、命令执行）或消耗大量 token 的输出，必须先确认。
